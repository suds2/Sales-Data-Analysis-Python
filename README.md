# 📊 Diwali Sales Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Diwali sales data using Python. The objective is to analyze customer purchasing behavior and identify valuable business insights that can help improve sales strategies.

## 🎯 Objectives
- Clean and preprocess the sales dataset.
- Analyze customer demographics.
- Identify top-performing states and cities.
- Find the most popular product categories.
- Study customer purchasing patterns.
- Generate meaningful business insights using visualizations.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## 📂 Dataset
- Diwali Sales Data.csv

## 📈 Key Insights
- Identified the age groups contributing the highest sales.
- Analyzed purchasing behavior based on gender.
- Compared sales across different states.
- Studied occupation-wise customer spending.
- Found top-selling product categories.

## 🚀 How to Run
1. Clone the repository.
2. Install the required libraries:
   ```
   pip install pandas numpy matplotlib
   ```
3. Open the notebook in Jupyter Notebook.
4. Run all cells.
